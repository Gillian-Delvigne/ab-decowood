=== Woodzone ===
Requires at least: 6.2
Tested up to: 6.6
Requires PHP: 7.4

== Description ==
Woodzone is a modern and clean WordPress theme designed for carpenters, craftsmen, woodworking projects, furniture repair and services, construction, handymen, and all types of carpentry businesses. Fully compatible with the WordPress Block Editor, this theme offers a professional design with a 100% responsive layout and is easy to customize, no coding skills needed.
